# iRays Vision

**Objective Brain Triage for the Field.**

iRays Vision is developing the first handheld device to replace the subjective penlight exam with a quantitative brain vital sign — delivering objective neurological assessment to prehospital providers in seconds.

## The Problem

EMS has real-time data for every critical organ except the brain. For cardiac emergencies there's the ECG. For respiratory distress, capnography. For neurological events — stroke, TBI, overdose — providers still rely on a penlight and a subjective guess.

## Technology

- **Dual-Spectrum Pupillometry:** Synchronized IR and visible-light capture of the pupillary light reflex.
- **Edge AI Processing:** On-device analysis with sub-10ms latency.
- **Quantitative Metrics:** Constriction velocity, dilation velocity, onset latency — delivered as a brain vital sign.
- **Ruggedized Hardware:** Custom PCB engineered for EMS duty cycles and field environments.
