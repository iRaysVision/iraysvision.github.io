# iRays Vision

**Dual-Spectrum Pupillometry for Prehospital Neuro-Triage.**

iRays Vision is developing a ruggedized, edge-AI medical device to provide objective neurological data to EMTs and Paramedics.

## Technology
* **Dual-Spectrum Imaging:** Synchronized IR and Visible light capture.
* **Edge AI:** On-device processing using Nvidia Jetson / STM32 for <10ms latency analysis.
* **Metrics:** Constriction Velocity, Dilation Velocity, and Onset Latency.

## Status
* **Pilot:** Currently piloting with Hampton Fire & Rescue.
* **IRB:** Exempt status granted for clinical data collection.
* **IP:** Patent Pending.
